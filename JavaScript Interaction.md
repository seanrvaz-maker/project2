# Sean Vaz Sports Media Portfolio

## Project Description
A responsive portfolio landing page showcasing my sports media projects, skills, and contact information.

## Intended User or Player
Sports employers, internship recruiters, and professors.

## User / Player Goal
Allow visitors to quickly learn about my experience and view my projects.

## Inspiration Interfaces
- Nike
- ESPN
- Adobe Portfolio

## Design Choices

Layout:
Single-page scrolling website.

Color:
Black, white, blue, and gold accents.

Typography:
Arial for readability.

Visual hierarchy:
Large hero section followed by organized content sections.

Feedback states:
Navigation links, cards, and button change appearance on hover.

Responsive design:
Cards stack vertically and navigation changes to a column on smaller screens.
